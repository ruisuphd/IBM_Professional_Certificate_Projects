# Developing AI Applications with Python and Flask

**Source repo:** https://github.com/ruisuphd/IBM_Developing_AI_Applications_with_Python_and_Flask  
**Programme:** IBM AI Engineering Professional Certificate (Coursera)  
**Stack:** Python 3, Flask, HTML, JavaScript, IBM Watson NLP

## Overview

Hands-on labs and a final project building AI-powered web applications using Flask. Topics include RESTful API design, Jinja2 templating, unit testing, and integrating IBM Watson Natural Language Understanding into a Flask backend.

## Setup

```bash
pip install flask ibm-watson
flask run
```

Open `http://127.0.0.1:5000` in your browser.
