# Image Captioning App with Gradio

**Source repo:** https://github.com/ruisuphd/IBM_Image_captioning_app_with_Gradio  
**Programme:** IBM AI Engineering Professional Certificate (Coursera)  
**Stack:** Python, Gradio, Hugging Face Transformers (`Salesforce/blip-image-captioning-base`)

## Overview

A Gradio web app that accepts an uploaded image and returns a natural-language caption generated by a pre-trained vision-language model (BLIP). Demonstrates loading and running inference on a Hugging Face model locally.

## Setup

```bash
pip install gradio transformers torch pillow
python image_captioning_app.py
```

Gradio will print a local URL. Upload any image to receive a caption.
